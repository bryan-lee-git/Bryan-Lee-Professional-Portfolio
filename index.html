Recent Web Development / Programming Projects


Most Recent: Star Trek "Clicky Game" w/ React

A Star Trek themed memory game developed with React, React Transition Group, CSS Animations, and the Bulma CSS framework. Utilizes react components (stateful and stateless), user events, etc.

learnToCode.js - Lesson Scraper

Find, view, and make notes on lessons, videos, and articles scraped from code learning sites (w3schools, coursera, codecademy, youtube, medium). Mongodb, Express, Handlebars, Mongoose, Cheerio, Datatables, Axios.

Road To Rome (Group Project 2)

An RPG called Road to Rome. You play a gladiator who must fight his way from the "minor leagues" up to the ultimate glory in Rome. Node JS, MySQL, Sequelize, Sass, Handlebars, Express, and Passport.

A1 Guitar Learner App (Group Project)

A-1 Guitar Learner provides anyone learning the guitar one place to go to find everything they need to begin playing the guitar. Includes tuner, metronome, tabs and chords via Songsterr API, lesson videos via YouTube API, User Auth.

LIRI.js

LIRI is a Node.js Language Interpretation and Recognition Interface using several node packages and APIs including URL Shortening, Spotify, Request, and Moment.

trainScheduler.js

A data persistent train scheduler database / arrival time calculator app utilizing Firebase, Bootstrap, and Moment.js.
giftastic.js

Gif search app employing the GIPHY API, localStorage. Populates gifs from user input in a clean + responsive interface.

weatherChecker.js
